AWS EC2 Instance with Apache Web Server

This simple website demonstrates deploying a static site on an AWS EC2 instance using Apache middleware. Follow the steps below to launch your own instance and deploy this website.

Step 1: Launch an EC2 Instance

  1. Sign in to the AWS Management Console
  2. Navigate to EC2 Dashboard
  3. Click "Launch Instance"
  4. Choose an Amazon Machine Image (AMI) - select Amazon Linux 2 or Ubuntu Server
  5. Select an instance type (t2.micro for free tier)
  6. Configure instance details (use defaults for simplicity)
  7. Add storage (default 8GB is fine)
  8. Add tags (optional but recommended, e.g., Name: Apache-Web-Server)
  9. Configure security group:
    • Type: HTTP, Protocol: TCP, Port Range: 80, Source: 0.0.0.0/0
    • Type: SSH, Protocol: TCP, Port Range: 22, Source: Your IP or 0.0.0.0/0
  10. Review and launch
  11. Create or select a key pair for SSH access

Step 2: Deploy Website using Apache

  1. Connect to your instance via SSH: ssh -i your-key.pem ec2-user@your-instance-public-ip
  2. Update the system: sudo yum update -y (Amazon Linux) or sudo apt update && sudo apt upgrade -y (Ubuntu)
  3. Install Apache:
    • Amazon Linux: sudo yum install httpd -y
    • Ubuntu: sudo apt install apache2 -y
  4. Start Apache:
    • Amazon Linux: sudo systemctl start httpd && sudo systemctl enable httpd
    • Ubuntu: sudo systemctl start apache2 && sudo systemctl enable apache2
  5. Upload your website files to /var/www/html/
  6. Set proper permissions: sudo chown -R apache:apache /var/www/html/ (Amazon Linux) or sudo chown -R www-data:www-data /var/www/html/ (Ubuntu)
  7. Access your website at the instance's public IP address

Key Points to Remember

  • Always stop your instances when not in use to avoid charges
  • Security groups control inbound traffic - keep them restrictive
  • Use AWS Free Tier resources to minimize costs while learning
  • Regularly backup your data and configurations
